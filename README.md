# drift-ctl-analysis

Analyze [drift-ctl](https://driftctl.com/) json file for quick report building.

# Getting Started 

## Pre Requisites 

1. Node.js 
1. NPM

## Getting Started 

1. Place the drift-ctl json file in data folder based on environment
2. In Terminal 

```sh
npm install 
npm run report <test | dev>
```
